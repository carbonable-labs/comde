## About

Pufu is a simple contract based on Compose-Decompose (Comde) module.

## Getting Started

### Prerequisites

- [Cairo](https://github.com/starkware-libs/cairo)

## Usage

### Build

```bash
scarb build
```

### Test

```bash
scarb test
```

### Format

```bash
scarb fmt
```
